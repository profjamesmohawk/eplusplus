Password Reset Application

You work for Acme Cogs and are considering the development of a self-serve password reset application that will reduce calls to the help desk.

Should you invest in this application?

The project will:
  • take 2 developers 6 months
  • require a $5,000 development server to be purchased before the project starts
  • require a $5,000 production server to be purchased before the app goes live
  • increase calls to the help desk by 100/month for the first two months of production
  • reduce calls to the help desk by 200/month for the next 34 months
  • servers cost $100/month to host
  • developers cost $80/hour
  • each help desk call costs $50

Classical financial math can be used to answer questions like this one, but few people do such math anymore. A much more common approach is to create a spreadsheet tracking all the costs and benefits, month by month.

Use any strategy you like to answer the question in your tutorial.
