What matters to you.

Today's exercise is intended to help you determine what is important to you. It comes from a program called Acceptance and Commitment Therapy developed by Russ Harris.
  • Watch this short video
  • Sort and group these value cards
  • It can also be useful to arrange value cards into a tree. Values that you want to achieve are the leaves and values that sustain you are the roots.
  • Note: This is an optional activity. There are no grades associated with it.
